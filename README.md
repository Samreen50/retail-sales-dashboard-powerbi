# 📊 Retail Sales Dashboard (Power BI)

## 🚀 Project Overview

This project presents a **Retail Sales Performance Dashboard** built using **Power BI**. The dashboard provides actionable insights into sales trends, customer behavior, and product performance to support data-driven decision-making.

---

## 🎯 Objectives

* Analyze overall sales performance
* Identify top-performing products and categories
* Track monthly and yearly sales trends
* Understand customer purchasing behavior
* Provide insights to improve business strategy

---

## 🛠️ Tools & Technologies Used

* Power BI
* Microsoft Excel (Data Source)
* Data Cleaning & Transformation (Power Query)

---

## 📂 Dataset

* Sales data including:

  * Order Date
  * Product Category
  * Sales Amount
  * Quantity
  * Customer Information

---

## 📈 Key Insights

* Identified top-performing product categories
* Analyzed seasonal sales trends
* Highlighted high-value customers
* Detected patterns in order frequency

---

## 📊 Dashboard Features

* Sales Overview (Total Sales, Profit, Quantity)
* Monthly Sales Trend Analysis
* Category-wise Performance
* Customer Segmentation
* Interactive Filters (Date, Category, Region)
